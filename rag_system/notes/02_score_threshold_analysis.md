# Score Threshold 사후 필터 분석

## 배경 및 동기

### 관찰된 사실
- EXP-02-003 (V1 프롬프트 + `--classifier none`): MAP 0.8318
- EXP-04-002 (V4 프롬프트 + 기본 classifier): MAP 0.7667
- Δ = +0.0651, classifier 자체가 순손해임을 시사

### 원인 (EXP-04-003 관찰 노트 참조)
- 문서(ko_mmlu + ko_ai2_arc)에 순수 과학 외에 사회·역사·경제·상식 문서 포함
- "과학 vs 비과학" 이분법 분류가 대회 도메인에 부적합
- V4가 거른 "비과학"으로 판정된 질문 중 상당수가 실제로는 문서 내에 답 존재

### 전략
- 분류기 제거 후 rerank score로 사후 필터링
- 모든 질의 검색 → rerank → top-1 score < τ 이면 `topk=[]` 처리
- τ 최적값을 찾기 위해 0.8318 제출 CSV를 분석
- 분석 스크립트: `rag_system/analyze_score_threshold.py`

## 해석

### 1. 분포가 명확히 이원화
- < 0.05: 23건 - 거의 순수 잡담/인사/AI 관련 질의
- 0.05~0.30: 11건 - 경계 (과학·잡담 혼재)
- > 0.30: 186건 - 대부분 실제 과학 질의, rerank가 고점수 매칭
- > 0.90 집중: 134건 (61%) - 아주 자신 있는 매칭

→ τ = 0.05 근처에 자연스러운 구분선 존재

### 2. "답변 불가" 마커는 threshold 판단에 부적합
39개 "답변 불가" 중 top-1 score가 0.7 이상인 케이스 다수:
- id=25 (금성 달, 0.957), id=221 (전구 병렬, 0.994)
- id=68 (연산자, 0.995), id=102 (기체 질량, 0.968)
- id=252 (해구, 0.920), id=21 (클래스, 0.903)

이는 rerank는 관련 문서를 잘 찾았지만 LLM이 보수적으로 답변 거부한 경우.
→ 답변 불가 ≠ retrieval 실패. 이 마커를 필터링 신호로 쓰면 안 됨.

### 3. τ = 0.05 리스크 분석
커트되는 23건 중 진짜 과학 질문일 가능성:
- id=93 "잠 잘 잤을 때 이로운 점" (0.0085) - 건강 일반 상식, 모호
- 나머지 22건: 명백한 잡담/감정/AI 질문

→ 오분류 리스크 최대 1~2건, 대부분 올바른 커트

### 4. τ 후보별 기대 MAP 이득

| τ | 커트 | 기대 Δ MAP | 판단 |
|---|-----|-----------|------|
| 0.05 | 23 | +0.06 ~ +0.08 | ✅ 안전, 리스크 낮음 |
| 0.08 | 26 | +0.07 내외 | ⚠ id=305(농장 영향)이 과학일 가능성 |
| 0.10 | 29 | 비슷 | 리스크↑ |


## 결정

τ = 0.05 채택

- 0.8318 → 예상 0.88 ~ 0.91
- 제출 1회로 대폭 개선 가능성
- 오분류 리스크 최소화

1. τ=0.05가 거의 이상적인 단절점 
2. 오분류 리스크도 거의 없음 - 과학 질문이 잘못 잘린 케이스는 305(농장 해안) 정도가 애매, 나머지는 명백한 잡담
3. 남은 개선 여지: 고점수 "답변 불가" 케이스 (14, 21, 25, 27, 102, 221, 252 등)는 retrieval은 관련 문서를 찾았으나 실제 GT와 불일치 → reranker 또는 embedding 단에서 개선 필요
